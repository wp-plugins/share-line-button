== Description ==

This plugin will create a Line button for share a content to Line's timeline or your frined in Line application

The button will create on mobile site only. It useless when click on pc browser.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
